# FPGA_ZYNQ_OLED_INTERFACING_USING_SPI_CONTROLLER
This project demonstrates OLED interfacing on the Zynq FPGA by utilizing both the Processing System (PS) and Programmable Logic (PL). The PS generates and sends customized messages through the SPI controller, while the PL assists in hardware interfacing and data transfer, enabling real-time communication and display of text on the OLED screen.
